# 内容密度参考标准

以下为缺省值，可按学科特性调整：

| 项 | 推荐下界 |
|---|---|
| 每章字数 | ≥ 3000 字（人文社科类可适当宽松，技术类不应低于此） |
| 例题数 | ≥ 3 个/章，须完整解答 |
| 习题数 | ≥ 5 道/章，至少 3 个难度等级 |
| 习题答案/提示 | 100% 覆盖，缺一不可 |
| 关键定理证明 | 100% 覆盖（完整证明或梗概） |
| 本章自检清单 | 3–7 条具体任务 |

## 学科调整建议

- **数学 / 理论计算机**：例题与证明权重高，单章可达 5000–8000 字；
- **编程 / 工程**：例题须为可运行代码，每章应包含至少 1 个完整项目片段；
- **实验科学**：例题中应有"数据 → 分析 → 结论"完整链路；
- **人文社科**：例题可改为"案例分析"，习题改为"论述题 + 参考答题框架"。
